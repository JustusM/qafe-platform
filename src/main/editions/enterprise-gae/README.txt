HOW TO INSTALL?
---------------
Unzip the war found in the zipfile.
The license key file must be stored in the WEB-INF directory of the unzipped QAFE directory.
After these steps the directory can be deployed on the Google App Engine.
For retrieving a license key file, please visit http://www.qafe.com/registration
